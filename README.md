# Bash Toolkit

A collection of useful bash scripts.

## Scripts

* `bq_table_copy.sh`: Copies a BigQuery table.
* `count_rips.sh`: Counts rips.
* `count_users_rips.sh`: Counts rips for each user.
* `extract_keys.sh`: Extracts keys from a file.
* `split_csv.sh`: Splits a CSV file into multiple files.
* `Makefile`: Makefile for the project.
